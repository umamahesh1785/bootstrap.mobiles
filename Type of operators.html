Data Types in Js

There are two types of DT:1-primitive
2-non primitive

1.primitive:it is immutable and it stores single value,types of primitive data types

  1. Numbers
  2. strings
  3. boolean
  4. null
  5. undefined
  6. symbol

Non-primitive DT:complex,stores multiple values, mutable,it returns object DT

  1. Object
  2. Arrays
  3. Date
